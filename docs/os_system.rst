os system 
========

Check ubuntu version
--------
:code:`$ cat /etc/os-release`


Check shell
--------
:code:`$ echo $SHELL`


Change shell to bash
--------
:code:`$ echo $SHELL`


Check IP address
--------
:code:`$ echo $SHELL`


CPU cores
--------
:code:`$ lscpu`


GPU info
--------
:code:`$ nvidia-smi`


Memory info
--------
:code:`$ free -h`


Disk partition usage
--------
:code:`$ df -h`


Shutdown system
--------
:code:`$ sudo halt`


Restart system
--------
:code:`$ sudo reboot`

conda env
========

List available envs
--------

:code:`$ conda info --envs`

Create env
--------

:code:`$ conda create --name <env_name> python=3.6`

Activate env
--------

:code:`$ source activate <env_name>`

Deactivate env
--------

:code:`$ source deactivate`

Remove env
--------

:code:`$ conda remove --name <env_name> --all`
